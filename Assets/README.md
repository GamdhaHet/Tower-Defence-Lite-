## Tower Defence (Lite)

Tower Defence (Lite) game project built with **Unity 2022.3.62f2**.

## Getting Started

1. Install Unity **2022.3.62f2**.
2. Open this project folder in Unity.
3. Open the main scene: `Assets/Scenes/SampleScene.unity`
4. Click **Play** in the Unity Editor.

## Project Layout (High Level)

- Game logic and systems live under `Assets/Script/`
- UI-related scripts live under `Assets/Script/UI/`
- Scene assets live under `Assets/Scenes/`
